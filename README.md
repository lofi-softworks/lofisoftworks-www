# LoFi Softworks — GitHub Pages Site (Minimal)

This repo contains a minimal 2-page landing site for **LoFi Softworks**:

- `index.html` — landing page + contact form
- `privacy.html` — privacy policy
- `styles.css` — styling
- `assets/` — logo + favicon

## Contact form

The contact form is handled by **Formspree** and posts to:

- https://formspree.io/f/xpqjzayo

Submissions are delivered to **lofisoftworks@gmail.com**.

## GitHub Pages setup

1. Push this repo to GitHub
2. Repo Settings → **Pages**
3. Deploy from **Branch**: `main` / `(root)`
